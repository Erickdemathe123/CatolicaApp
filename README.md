Escopo
O App do Católica SC serve para ver notas/faltas, ver notícias, ver número da sala, ver dias de aula, olhar seu perfil e muito mais

Tela inicial:
-Login.
-Cadastro.

Tela 2
-Notícias.
-Avançar para o Menu.

Tela 3
-Ver Mensagens.
-Ver Perfil.
-Ver Agenda.
-Ver Disciplinas.
-Ver Históricos.	
-Ver Horários.
-Ver Notas/Faltas.
-Ver Solicitações.

Tela 4
-Selecionar o período.

Tela 5
-Ver o Detalhes do que Selecionou
